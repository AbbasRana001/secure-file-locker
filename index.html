How the Vigenère cipher works

1

Input text

Each letter maps to a number (A=0, B=1 … Z=25). Non-letters pass through unchanged.

2

Repeat key

The keyword cycles to match the length of your text, character by character.

3

Shift & wrap

Each letter is shifted by its key value, wrapping around at 26.

Encrypt: Eᵢ = (Pᵢ + Kᵢ) mod 26
Decrypt: Dᵢ = (Cᵢ − Kᵢ + 26) mod 26
